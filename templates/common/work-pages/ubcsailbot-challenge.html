UBC Sailbot - International Robotic Sailing Regatta

In 2012-2013 I was the control team lead for the UBC Sailbot team. The team designed a new autonomous sailbot from scratch over the course of the 2012-2013 school year. We built an entirely new control system using python on a Raspberry Pi. We connected the system to a low level real time unit (an arduino) to connect to sensors and run some of our real time processes.

The code we built is all open source and was entirely developed handful of UBC engineering students and me. We are currently using what we learned to continue with an entirely new system for the Microtransat challenge.

We competed in the 2013 International Robotic Sailing Regatta hosted by Olin College in Gloucester, Massachusetts and took first place with the first ever perfect 50/50 score.
